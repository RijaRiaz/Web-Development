ID and Classes

we can assign one ID to one tag only. While we can assign one class name to multiple tags.

Div and Span

We use div as block element.

We use span for inline elements for example to style a specific word inside a div.

Floating

Elements can float horizontally and not vertically. For this we use properties, float: left; and clear: both;

Fixed Positioning

Fixed on the screen and does not move when you scroll the window.

Relative Positioning

Change the position relative to its original position.

Absolute Positioning

Change the position relative to the original page. We can place element anywhere on the page with it. Depend on parent.

Display

If we set visibility to hidden, the block will hide but space occupy by it will remain oppocupied as empty. If we use Display: none; the space will not be occupied and the block will remove.
